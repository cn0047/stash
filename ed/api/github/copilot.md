Copilot
-

````sh
# slash commands:
/clear    # clear conversation.
/delete   # delete a conversation.
/new      # start new conversation.
/doc      # add comments for specified or selected code.
/explain  # get code explanations.
/fix      # propose fix for problems in selected code.
/generate # generate code to answer specified question.
/optimize # improve runtime of selected code.
/tests    # create unit tests for selected code.
````
