## Cloud Native

Being Cloud Native means to satisfy and follow next requirements:

* Microservices architecture + Gateway.
* Containers + Container orchestrator.
* DevOps (CI/CD).
* Cloud Native Open Standards (logging, tracing, service discovery, etc.).
